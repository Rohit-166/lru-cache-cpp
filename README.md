# LRU Cache (C++)

## Description
Implemented an LRU Cache from scratch in C++ using a custom Hash Table and Doubly Linked List.

## Features
- O(1) get and put operations
- Manual memory management
- No STL containers used

## Concepts Used
- Hashing
- Doubly Linked List
- Pointer manipulation

## Complexity
- Time: O(1) per operation
- Space: O(capacity)
